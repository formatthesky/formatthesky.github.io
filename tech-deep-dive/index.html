Background

Format The Sky is not just a band; it's a living digital ecosystem. To maintain a consistent, high-frequency presence on short-form video platforms (TikTok, IG Reels, YT Shorts) without burning out the human producer, we built the AGE Pipeline (Automated Glitch Era).

The Workflow: From Audio to Asset

1. Section Inference (The "Pulse")

The pipeline starts by analyzing the raw audio of our tracks. We use an LLM-based inference system to identify the song's structure:

  • Timestamping: Chorus (Catchy/High Energy), Verse (Atmospheric/Low Energy), Breakdown (Intense/Chaotic).
  • BPM Sync: All visual pulses are calculated from the song's BPM to ensure perfect rhythmic alignment.

2. CLIP Indexing (Semantic Discovery)

We maintain a library of 100+ raw high-quality 4k video clips. These are pre-processed through OpenAI’s CLIP model to create a searchable semantic index using cosine similarity for high-precision visual retrieval.

  • Searchable Attributes: "Neon rain," "Cybernetic eye," "Analog glitch," "Chrome textures."
  • AssetLibrary: A custom TypeScript class that manages these embeddings and allows for rapid visual retrieval.

3. SYSTEM_SYNTHESIS: Applying "The Erasure"

This is where the latent space is shaped into our visual reality. We use ComfyUI workflows (via its API) to apply "The Erasure" aesthetic.

  • Image-to-Image / Video-to-Video: We take the indexed clips and run them through custom nodes (ControlNet, IP-Adapter, AnimateDiff).
  • Style Injection: We inject "Format The Sky" brand elements (magenta/cyan color palettes, CRT scanlines, glitch displacement) into every frame.
  • Model Orchestration: We use VramOrchestrator to swap between Flux (for high-fidelity stills) and SDXL (for fast video processing).

4. Hybrid Vision QC (The "Gatekeeper")

Automation requires accountability. Before any asset is finalized, it must pass a dual-model Quality Control check to ensure it meets our "Beauty + Threat" brand criteria:

  • Gemma 3: Generates a frame-by-frame objective description.
  • Qwen 2.5: Acts as a "Devil's Advocate," reviewing the description for static imagery, brand inconsistency, or technical glitches that don't look "intentional."
